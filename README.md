tested on debian 10
```
wget --no-check-certificate -O snell.sh https://raw.githubusercontent.com/ascfin/snell.sh/master/snell.sh
chmod +x snell.sh
./snell.sh
```
```
vim /etc/snell/snell-server.conf
systemctl status snell
```
how to uninstall
```
wget --no-check-certificate -O uninstall-snell.sh https://raw.githubusercontent.com/ascfin/snell.sh/master/uninstall-snell.sh
chmod +x uninstall-snell.sh
./uninstall-snell.sh
```
